# customer-churn-prediction
Machine Learning project to predict customer churn based on historical data. The model analyzes customer behavior and identifies key factors influencing churn using techniques like Logistic Regression, Random Forest, and XGBoost. Includes data preprocessing, EDA, feature engineering, model evaluation, and SHAP-based explainability.
